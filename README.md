# Portfolio Website

This is a responsive personal portfolio website built with plain HTML, CSS, and JavaScript.

## Files
- `index.html` — main page structure
- `styles.css` — visual design and responsive layout
- `script.js` — mobile menu toggle and footer year

## How to deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Open the repository settings.
4. Go to Pages and choose the branch to deploy from (usually `main`).
5. Save the settings. GitHub Pages will publish the site.

## Local preview
Open `index.html` in a browser, or run:

```bash
python -m http.server 3000
```

Then visit: http://127.0.0.1:3000/
